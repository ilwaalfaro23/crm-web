# CRM Web Application

A simple web-based Customer Relationship Management (CRM) system
built with PHP Native and MySQL.

## Features
- Admin authentication (login & logout)
- Customer management (CRUD)
- Customer follow-up history
- Search customer
- Dashboard statistics
- Responsive admin dashboard

## Tech Stack
- PHP Native
- MySQL
- Bootstrap 5

## Installation
1. Clone repository
2. Import database `crm_db`
3. Set database connection in `config/database.php`
4. Run on XAMPP / Apache
